# Retail Sales Intelligence System


## Overview
This project is an end-to-end retail data analytics and inventory monitoring system built using Python, PostgreSQL, and Power BI.
The system extracts product data from an external API, stores it in a PostgreSQL database, performs inventory monitoring, sends automated low-stock email alerts, and visualizes business insights through an interactive Power BI dashboard.

## Features
- API-based retail product data extraction
- Automated ETL pipeline using Python
- PostgreSQL database integration
- Inventory monitoring system
- Automated low-stock email alerts
- Interactive Power BI dashboard
- Category and price filtering
- Conditional formatting for inventory risk detection
- Environment variable security using .env


- ## Tech Stack
- Python
- PostgreSQL
- SQLAlchemy
- Pandas
- Power BI
- SMTP Email Automation
- REST API
- dotenv

## ETL Pipeline
1. Product data is extracted from an external API
2. Data is transformed using Pandas
3. Cleaned data is loaded into PostgreSQL
4. Inventory monitoring scripts analyze stock levels
5. Automated email alerts are triggered for low inventory items
6. Power BI connects directly to PostgreSQL for visualization


## Power BI Dashboard Insights
The dashboard provides:
- Inventory monitoring by product
- Conditional low-stock highlighting
- Top-rated product analysis
- Category-based filtering
- Price-range filtering
- Interactive inventory exploration
